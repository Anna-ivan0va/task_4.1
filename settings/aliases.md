# Псевдонимы

Они созданы для того, чтобы не печатать каждую команду git целиком

---
**git config alias.SHORT_NAME *[имя для сокращения]* COMMAND *[команда, которую нужно сократить]***

---

## Принятые сокращения:

- st = status
- ch = checkout
- br = branch
- mg = merge
- cm = commit
- reb = rebase
- lg = «git log --pretty=format:'%h — %ar: %s'»
